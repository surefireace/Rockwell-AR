This method is commonly used to instantiate player characters. If a match has to be started \char`\"{}actively\char`\"{}, you can call an Pun\+R\+PC triggered by a user\textquotesingle{}s button-\/press or a timer.

When this is called, you can usually already access the existing players in the room via Photon\+Network.\+Player\+List. Also, all custom properties should be already available as Room.\+custom\+Properties.
